Язык: Python
Библиотеки: Selenium, BeautifulSoup4, fake_useragent, requests, csv, re, os, random
Дополнительно: chromedriver.exe

Суть подхода к GET запросам на яндекс маркет:
    Необходимо заполнить proxy_bank стабильными рабочими прокси,
    чем больше, тем лучше. [Потенциально] Каждый GET запрос на яндекс маркет
    реализуется со случайным IP.
    Случайность IP реализуется за счёт рандомного выбора прокси сервера 
    из proxy_bank.

--UPD:
    Немного перестроена структура вызова функций,
    Убрал необходимость 6-этажного вызова функций:
        1) ya_get(url) получает url на страницу категории товаров,
        возвращает список url на каждый товар на страницу

        2) parse_page(url) получает url на 1 товар и парсит страницу на данные, скачивает картинки

        3) [Временно] окна selenium работают в активном режиме, постоянно нужно нажимать кнопку 
        "Я не робот".
    
        Пункт '3)' решается с использованием прокси + существенно возрастёт скорость, т.к.
        Можно будет удалить искусственные задержки, которые нужны, чтобы успеть нажать кнопку
